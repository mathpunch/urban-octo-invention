### 4. Push to GitHub
```bash
git init
git add .
git commit -m "Zyron 2.0 initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/zyron-2.0.git
git push -u origin main
```

### 5. Deploy on Vercel
1. Go to https://vercel.com
2. Click **"Add New Project"**
3. Import your GitHub repo
4. Leave all settings as default (Vercel auto-detects Node.js)
5. Click **Deploy**
